# Blinkit Sales Dashboard (Excel)

## Project Overview
This project analyzes Blinkit sales data using Microsoft Excel.

## Features
- Total Sales, Avg Sales, Ratings
- Outlet Size & Location Analysis
- Product Category Insights
- Interactive Filters using Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Insights
- Tier 1 locations generate highest revenue
- Fruits & Snacks are top-selling categories
- Medium outlets perform consistently well
